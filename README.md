# Hotel Booking Data Analysis (Python)

This project analyzes hotel booking data to find key business insights.

## Key Analysis:
- Total revenue calculation
- City vs Resort revenue comparison
- Total bookings and cancellations
- Monthly booking trends
- Monthly cancellation trends
- Monthly cancellation rate

## Key Insights:
- Highest booking month: August
- Highest cancellation month: August
- Highest cancellation rate month: June

## Purpose:
This project helps hotel owners understand their data clearly and make better business decisions.

## Tools Used:
- Python
- CSV file handling
- Data analysis logic
